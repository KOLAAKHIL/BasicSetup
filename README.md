# BasicSetup

Android interview starter project using Kotlin and Jetpack Compose.

## Included stack

- Jetpack Compose + Material 3 for UI
- Lifecycle runtime, Compose lifecycle, and ViewModel support
- Navigation Compose for multi-screen assignments
- Kotlin coroutines and Flow testing helpers
- Retrofit, OkHttp logging, and kotlinx.serialization for API work
- Room with KSP for local database tasks
- Coil Compose for image loading
- Koin for lightweight dependency injection
- JUnit, MockK, Turbine, Espresso, and Compose UI testing

## Useful commands

```bash
./gradlew :app:assembleDebug
./gradlew :app:testDebugUnitTest
```

For a full quick check:

```bash
./gradlew :app:testDebugUnitTest :app:assembleDebug
```
